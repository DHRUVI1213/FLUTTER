 # Student TODO App

A clean, modern, and dynamic Flutter TODO application designed for busy students to manage daily activities such as assignments, study hours, and personal goals.

---

# This app demonstrates:

State management using setState()
Dynamic task rendering with ListView.builder
Real-time UI updates
Task categorization
Add & delete actions with smooth UI

---

 ## Features
 # Add Tasks
 
Easily add assignments, study tasks, and personal goals.
Category Selection
Choose from:
Assignment
Study
Personal
Remove Tasks
Swipe left or tap the delete icon to remove tasks instantly.
 Real-time Update
UI updates immediately using setState().
 Modern UI
Soft shadows
Rounded cards
Category-based color coding
Clean and minimalist Material 3 design
